Controls

What each control does

Rows / Cols
Grid size. More = smaller circles and tighter pattern; fewer = bigger circles.
Fill density
How much of the grid is filled (%). Higher = more connected blobs.
Margin %
Space around the grid. Higher = smaller drawing area, more border.
Background
Canvas background color. Does not change the composition.
Dot 1–5
Colors for each layer. Changing these only changes color; the pattern stays the same.
Stroke weight
Outline thickness on shapes. 0 = no outline.
Random seed
Number that determines the pattern. Same seed = same composition every time.
